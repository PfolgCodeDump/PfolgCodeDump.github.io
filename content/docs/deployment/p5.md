## 第五部分：自定义与扩展

### 5.1 项目结构

```
FunRadiusP/
├── app/              # 页面路由
├── components/       # React 组件
├── lib/              # 工具函数
├── content/          # 内容文件
├── scripts/          # 构建脚本
└── public/           # 静态资源
```

### 5.2 修改主题

#### 方式一：环境变量

```env
NEXT_PUBLIC_DEFAULT_PRIMARY_COLOR=#3b82f6
```

#### 方式二：CSS 变量

编辑 `app/globals.css`：

```css
:root {
  --primary: #3b82f6;
  --secondary: #2563eb;
}
```

### 5.3 修改导航栏

编辑 `components/Header.tsx`：

- 修改导航链接
- 修改 Logo
- 调整样式

### 5.4 修改页脚

编辑 `components/Footer.tsx`：

- 修改版权信息
- 添加友链
- 添加社交链接

### 5.5 添加新页面

1. 在 `app/` 下创建文件夹和 `page.tsx`：

```typescript
export default function CustomPage() {
  return (
    <div className="container mx-auto px-4 py-8">
      <h1>自定义页面</h1>
    </div>
  );
}
```

2. 在 `Header.tsx` 中添加导航链接。

### 5.6 修改粒子特效

编辑 `components/Particles.tsx` 或通过环境变量配置。

### 5.7 修改背景

编辑 `app/globals.css` 中的背景样式。

### 5.8 添加新功能

1. 在 `components/` 创建新组件
2. 在页面中引入使用
3. 如需配置，添加环境变量

### 5.9 代码修改建议

- 使用 Git 进行版本控制
- 修改前先 commit
- 小步迭代，逐步验证
